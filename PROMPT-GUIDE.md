## 🎯 Formula: Apa yang Bikin Prompt Bagus

```
[Subjek Detail] + [Gaya Visual] + [Pencahayaan] + [Kualitas]
```

### Sweet Spot: 30-80 kata
- Terlalu pendek (< 10): hasil random
- Terlalu panjang (> 100): model bingung
- Ideal: **30-80 kata** yang spesifik

### 2. Starting Patterns
| Pattern | Contoh |
|---------|--------|
| Direct description | `A hyper-realistic photo of...` |
| Action verb | `Create a dramatic portrait of...` |
| Role play | `Act as a professional photographer...` |

### 3. Structure Patterns

**Pattern A: Photography**
```
[Style] photo of [subject], [lighting], [mood], [quality]
```
Contoh:
```
Paparazzi-style extreme close-up photo of a woman, 
caught off-guard while turning toward the camera. 
Strong harsh on-camera flash, grainy high-ISO, 
raw candid street-photography feel. Ultra-realistic, 
cinematic realism, high detail skin texture.
```

**Pattern B: Product/Brand**
```
[Role] creating [output type] for [brand/product]. 
[Style details]. [Technical specs].
```
Contoh:
```
Act as a High-End Product Photographer and CGI Artist. 
Generate a single, minimalist, and surreal image where 
a cloud is shaped like [brand logo]. Clean white background, 
studio lighting, 8k resolution.
```

**Pattern C: Illustration/Art**
```
[Art style] of [subject], [color palette], [composition], 
[mood], [technical specs]
```
Contoh:
```
inspired by a classic pokemon gameboy screenshot but 
it's highly detailed beautiful pixel art, include cool 
effects from moves used, modern minimal ui
```

**Pattern D: Infographic**
```
[Output type] of [subject], [layout], [elements], 
[style], [specs]
```
Contoh:
```
A hyper-realistic 3D travel guide infographic poster 
for [country]. The country shape is rendered as a raised, 
textured relief map. Clean vertical composition, 
vibrant colors, 1080x1080.
```

## 🔧 Template: Tinggal Ganti [Bracket]

### Portrait
```
[Style] portrait of [subject], [lighting description], 
[expression/mood], [clothing/accessories], 
[background], [quality keywords]
```

### Landscape
```
[Time of day] landscape of [location], [weather/atmosphere], 
[foreground elements], [lighting], [mood], [quality]
```

### Product
```
[Style] product photography of [item], [surface/background], 
[lighting setup], [shadow style], [composition], [quality]
```

### Food
```
[Style] food photography of [dish], [plating description], 
[ingredients visible], [lighting], [atmosphere], [quality]
```

### Abstract/Art
```
[Art style] of [subject/concept], [color palette], 
[texture/material], [composition], [mood], [resolution]
```

## ❌ Common Mistakes

### Terlalu Vague
❌ `beautiful image of a mountain`
✅ `Mountain landscape at sunrise, golden light hitting snow-capped peaks, misty valley, pine trees in foreground, ultra realistic photography, 8k`

### Terlalu Banyak Keywords
❌ `realistic, detailed, 8k, 4k, cinematic, dramatic, moody, vibrant, elegant, modern, clean, professional, studio, editorial, hyper-realistic, photorealistic`
✅ `cinematic, ultra realistic, 8k resolution`

### Gak Ada Lighting
❌ `portrait of a woman`
✅ `portrait of a woman by window, soft natural light, warm skin tones`

### Gak Ada Mood/Atmosphere
❌ `city at night`
✅ `cyberpunk city at night, neon signs, wet reflections, moody atmosphere`

## 🎨 Cheat Sheet

### Lighting Keywords
- `golden hour` — warm, sunset/sunrise
- `dramatic lighting` — high contrast
- `soft natural light` — window, diffused
- `studio lighting` — controlled, professional
- `neon lighting` — colorful, cyberpunk
- `backlit` — silhouette, halo effect

### Mood Keywords
- `moody` — dark, atmospheric
- `vibrant` — colorful, energetic
- `minimalist` — clean, simple
- `elegant` — sophisticated, luxury
- `dramatic` — intense, cinematic
- `serene` — calm, peaceful

### Quality Keywords (pick 1-2)
- `ultra realistic` — photo-real
- `cinematic` — movie-like
- `8k resolution` — high detail
- `studio quality` — professional
- `editorial` — magazine style
- `clean composition` — balanced

## 📱 Nano Banana vs ChatGPT Image

### Nano Banana (Gemini)
- Lebih natural, photorealistic
- Bagus untuk: portrait, landscape, product
- Prompt: **deskriptif, detail-oriented**
- Panjang ideal: 30-80 kata

### ChatGPT Image
- Lebih artistik, stylized
- Bagus untuk: illustration, abstract, creative
- Prompt: **bisa lebih longgar, creative**
- Panjang ideal: 20-60 kata

2. Pilih template di atas
3. Isi [bracket] dengan ide lu
4. Tambah 1-2 quality keywords
5. Paste ke Nano Banana
6. Generate → lihat hasil → adjust
